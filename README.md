# SEO Landing Pages

Create and manage your SEO landing pages with ease.

## Installation

Add this line to your application's Gemfile:

    gem 'seo_landing_pages'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install seo_landing_pages
